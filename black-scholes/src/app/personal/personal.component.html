Projects Overview

local_atmBlack Scholes Plus

Purpose: Estimate the value of stock options for given stocks, at a desired expiration date and strike price.

Usage: User inputs the stock tickers they are interested in, as well as expiration dates (one for all tickers / one for each ticker / range) and strike prices (at the money / above by x units / above by x percent).
              The script then gathers all options that match these criteria, and performa Black Scholes calculations, and data is returned in a pandas DataFrame.

Used Technology: Angular11, Python3, numpy, pandas, scipy, yahoofinance, flask,

Check out the Black-Scholes tab for a demo!

gamesSpigot Hue Light Plugin

Purpose: A minecraft plugin, written in java, allowing the user to operate real-world Philips Hue Light bulbs with in game triggers

Usage: A player in minecraft may use a customized block to assign levers/buttons/lanterns to connect with an exising Hue Bridge, and can then operate lights with in game controls.

Used Technology: Java8, Maven, Lombok, FasterXml Jackson, Spigot, Phillips Hue

emoji_objectsLove Lights

emoji_peopleKinect Gesture Recognition

Purpose: Allow an xbox kinect v1 to trigger a request over an http server when a preconfigured gesture is recognized

Usage: User defines gesture by entering "training" mode, and standing in the gesture position for about 1 minute, then in various other positions around the area for another 2 minutes. This data is stored as "Skeletons", and tensorflow is used to develop a categorization of "gesture" or "not gesture". When gesture is triggered, an http request is sent.

Used Technology: C#, Kinect V1 SDK, Python3 tensorflow
